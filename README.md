# Mehmet Emin Deniz - Portfolyo Sitesi

Bu proje, **Mehmet Emin Deniz** için hazırlanmış, statik (HTML/CSS/JS) bir portfolyo sitesidir.  
GitHub Pages üzerinde kolayca yayınlanabilecek şekilde tasarlanmıştır.

## Sayfalar

- **`index.html`**: Anasayfa, kısa tanıtım ve öne çıkan projeler.
- **`about.html`**: Hakkımda bölümü, üniversite ve bölüm bilgileri.
- **`projects.html`**: Projelerin listelendiği sayfa.
- **`blog.html`**: Blog yazılarının bulunduğu sayfa (HTML içinden kolayca yeni yazı eklenebilir).
- **`contact.html`**: İletişim formu ve e-posta bilgisi.

## Tasarım

- Açık mavi tonlarda, modern ve sade bir arayüz.
- `css/style.css` dosyasında tüm stiller bulunur.
- Mobil, tablet ve masaüstü için **responsive** tasarım.

## Blog'a Yeni Yazı Ekleme

`blog.html` dosyasında açıklama yorum satırı içinde bir örnek şablon vardır:

1. `blog.html` dosyasını aç.
2. `<!-- ÖRNEK ŞABLON: Yeni yazı eklemek için kopyalayabilirsin -->` kısmındaki
   `<article class="blog-post">` bloğunu kopyala.
3. Başlık, tarih ve paragraf içeriklerini kendine göre düzenle.
4. Dosyayı kaydet.

Her yeni yazı, bir tane daha `<article class="blog-post">...</article>` bloğu ekleyerek oluşturulabilir.

## GitHub Pages ile Yayınlama

1. Bu klasörü bir GitHub reposu olarak oluştur.
2. `index.html` ana dizinde kalmalı.
3. GitHub repo ayarlarından **Pages** bölümüne gir.
4. Source olarak `main` (veya varsayılan branch) ve `/ (root)` klasörünü seç.
5. Kaydettiğinde birkaç dakika içinde site yayına alınır.
